# Atlasctl Swarm Spec Rails

This directory is the durable, repo-owned source of truth for specification rails.

## Ownership

Owned by this system:
- Proposals / PRDs
- Specs
- ADRs
- Lane trackers and implementation plans
- Support claim maps
- Policy references
- Closeouts

Awareness-only external namespaces:
- `.codex/`
- `.spec/`
- `.claude/`
- `.jules/`

This system does not own or mutate external tool/session state.
